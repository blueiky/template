## 通用型后台管理界面模板

采用 Vite3.x + React18 + React Router 6 + React Redux 构建

UI 组件库采用 Antd

### 简要描述

实现对后台管理界面的统一搭建以及可实现动态菜单及路由的渲染。

项目借鉴于 Arco Design Pro 项目的构建方式，移除了国际化的操作，简易化了权限菜单的渲染逻辑

本项目已默认安装 `axios`，`axios` 的二次封装，由开发者自行二次封装

#### 开发

**依赖安装**
```shell
$ yarn 
```

**项目运行**
```shell
$ yarn run dev
```

**构建**
```shell
$ yarn run build
```

### 最后

本项目仅做基本内容框架构建
